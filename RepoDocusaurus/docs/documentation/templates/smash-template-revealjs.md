Reveal.js template.

## 安装

```bash
# 步骤 1 下载模板
$ mkdir HelloRevealJs && cd HelloRevealJs
$ smash download smash-template-revealjs

# 步骤 2 安装项目依赖
$ npm install

# 步骤 3 安装中间件
$ smash install
```

## 开发

```bash
$ smash run dev
```

## 构建

```bash
$ smash run build
```
